⚡ Torrent Guide

Some games will need a torrent and will be written with (Torrent Required) next to them.

Here is the website for a torrent client: https://webtorrent.io/

Scroll down the website and press "Get for Desktop".

Guide on How to Use Torrent:

  1. Download the torrent client from the link above.
  2. Paste the game file into the torrent (only for torrent-required games).
  3. Once downloaded, the game will be split into parts. Find the file called setup.exe.
  4. Run setup.exe. If it asks to make a shortcut or install add-ons, check those options. Everything else is set automatically.
